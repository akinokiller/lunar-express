## [1.1.0] - 2026-07-22 — 手感与平衡性调优(commit 58cfcef)

玩家试玩反馈:能源消耗太少、货物损伤太大、终点交货也掉血。

### 能源系统:从摆设变成策略点

| 参数 | 旧值 | 新值 |
|---|---|---|
| 油门耗电 `throttlePerSec` | 1.9 /s | **5.7 /s** |
| 滑行回充 `coastRegenPerSec` | 1.1 /s | **0.4 /s** |
| 电池布局 | L1×3 / L2×4 / L3×5 | **L1×1 / L2×2 / L3×6** |

- L1 仅 1 块电池是有意设计:作为能源教学关,迫使玩家第一次思考"何时给油、下坡滑行回充、绕路捡电池"。
- 检查点复活保底 30 能源,避免一次失误导致死循环。

### 货物损伤:整体温和化

| 参数 | 旧值 | 新值 |
|---|---|---|
| 无伤阈值 `safeImpact` | 5 m/s | **6 m/s** |
| 中伤上限 `midMaxDamage` | 25 | **18** |
| 重击系数 `heavyExtraPerMS` | 8 | **5** |
| 重击封顶 `heavyCap` | 60 | **45** |
| 翻车基础伤害 `roofBase` | 14 | **10** |
| 障碍伤害(基础/系数/封顶) | 8 / 1.6 / 24 | **5.5 / 0.9 / 14**(约降 1/3) |

### 终点交货:保护区免伤

- 新增 `FINISH_PROTECT_DIST = 18`(m):终点前 18 米为"基地磁力软着陆保护区",区内冲击、翻车、障碍碰撞均不伤货物。
- 高速冲入终点的弹回惩罚保留,但弹回本身及区内落地不再扣货(移除 `finishDamage` 判定)。

### 仿真验证(AI 驾驶员通关实测)

| 关卡 | 货物完整度 | 剩余能源 |
|---|---|---|
| L1 静海首航 | 100% | 37% |
| L2 风暴洋裂谷 | 86% | 37% |
| L3 第谷环形山 | 67% | 11% |

### 测试

- 24 项全部通过:更新伤害公式与终点弹回断言;新增"保护区内重落地不伤货物 / 区外对照仍伤货"用例;AI 冒烟增加每关货物/能源目标区间断言。

## [1.0.0] - 2026-07-21 — 首次发布(commit 2129029)

完整可玩的原创浏览器游戏,Canvas 2D + TypeScript + Vite,无后端依赖。

- 低重力(2.6 m/s²)月球快递:油门/刹车/空中姿态,双轮弹簧悬挂刚体物理
- 3 个渐进关卡:静海首航 → 风暴洋裂谷 → 第谷环形山
- 坡道、坑洞、浮沉无人机与弹跳滚石障碍、检查点复活(+15s / -150 分)
- HUD:速度、能源、货物完整度、用时、实时得分、关卡进度
- 完整流程:标题选关 → 新手引导(可跳过)→ 暂停 → 失败/过关结算 → 三关总评(S/A/B/C)
- Web Audio 程序合成音效(引擎轰鸣随油门变调),开关持久化
- localStorage 存档:每关最高分、已解锁关卡、音效开关、引导已读
- 复古科幻海报 × 扁平插画视觉:四层视差背景、粒子系统、摄像机平滑跟随
- 移动端触控按钮,响应式适配,DPR 缩放
- 部署:Cloudflare Pages(直接上传)→ 自定义域名 lunar.dengdeng.uk
- 初始平衡:无伤阈值 5 m/s(由 4 上调,仿真表明 4 过苛);滚石为"腾空从下方通过"的时机闸门设计
